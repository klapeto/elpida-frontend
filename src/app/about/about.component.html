About Elpida

Elpida was started as open and free benchmarking framework aiming to provide a transparent way to benchmark algorithms/systems with no closed source components. This way everything is benchmarked on equal ground with no result skews. The website/Http Api was created to store anonymously results so that users can compare similar systems as well as create statistics.

Privacy Disclaimer

We respect freedom and privacy. For this we do not store personal information anywhere. Results uploaded from the native application are anonymous and can be summarized in:

  • Timestamp of the result
  • Elpida version and compiler that built it
  • System information including:
    • CPUID information including caches information
    • Operating system information including category (eg. Windows/Linux), name(eg. Windows 10 Pro/Ubuntu) and version(eg. 2004/ 18.04)
    • Main memory information (Total size, page size)
    • Topology and affinity used by the benchmark
  • Benchmark information including the input data size used

If you think the above information can identify you, you can always opt out from uploading by unchecking the option in the native application before start benchmarking. The option is very visible just above the 'Start' button so it is difficult to miss that.

You can always review the code all applications to review what data is sent via the repositories.

Both this website and Http Api server do not keep cookies. We do not care who you are, we do not need them, we do not use them.
